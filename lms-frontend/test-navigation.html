🎓 LMS Frontend Navigation Test Guide

✅ Step 1: Access the Application

Open your browser and navigate to: http://localhost:3002

You should see the LMS login page with demo account credentials displayed.

📋 Step 2: Login as Student

Use the following credentials to test the student experience:

  • Username: student1
  • Password: admin123

After login, you'll be redirected to the dashboard.

🔍 Step 3: Browse Courses

From the left sidebar, click on "Browse Courses" or navigate to:

http://localhost:3002/courses

Expected Result:

  • You should see a course listing page with search and filter options
  • Two courses should be visible: "Java Full Stack" and "Python Full Stack"
  • Each course shows title, description, instructor, lessons, and an "Enroll" button
  • Search functionality should work in real-time

🎯 Step 4: Enroll in a Course

On the Browse Courses page:

  1. Click the "Enroll" button on any course
  2. You should see a success toast notification
  3. The button should change to "Enrolled" and become disabled
  4. Click "View Details" to see the full course page

📚 Step 5: View My Enrollments

From the left sidebar, click on "My Enrollments" or navigate to:

http://localhost:3002/my-enrollments

Expected Result:

  • You should see all courses you've enrolled in
  • Each enrollment shows progress percentage and completion status
  • Progress bars indicate learning progress
  • "Continue Learning" buttons link back to course details

🔍 Step 6: Course Details & Progress

Click on any course (either from Browse or My Enrollments):

  1. You should see the detailed course view with lessons
  2. For enrolled courses, you can mark lessons as complete
  3. Progress percentage updates as you complete lessons
  4. Access to discussions, reviews, and certificate (when 100% complete)

🔧 Troubleshooting

If you encounter any issues:

  • Pages not loading: Check that both frontend (port 3002) and backend (port 8080) are running
  • Login fails: Verify backend is connected to MySQL and user data exists
  • Courses not showing: Ensure courses are published (status = PUBLISHED)
  • Navigation broken: Clear browser cache and refresh

🎉 Success Criteria

The implementation is working correctly if you can:

  • ✅ Login successfully with student credentials
  • ✅ Navigate to Browse Courses and see course listings
  • ✅ Use search and filter functionality
  • ✅ Enroll in courses with one click
  • ✅ Access My Enrollments and see enrolled courses
  • ✅ Navigate between pages using the sidebar
  • ✅ View detailed course information and track progress
